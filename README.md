# IMDB Sentiment Classification

This project focuses on binary sentiment classification using the IMDB movie review dataset.

## Project Goal
The aim of this project is to classify movie reviews as positive or negative and compare different methodologies covered in the course.

## Methods Applied
- Logistic Regression
- Basic MLP
- MLP + Dropout
- MLP + L2 Regularization
- MLP + Early Stopping
- Optimizer Comparison (Adam vs SGD)
- MLP with BatchNorm and He Initialization

## Dataset
IMDB movie reviews dataset

## Task
Binary classification:
- 0 = negative
- 1 = positive

## Best Model
MLP + Dropout + Early Stopping
